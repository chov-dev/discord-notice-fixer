# 📌 Discord Notice Fixer
> 📒 소개

채팅창 하단에 공지를 고정하여 주는 봇입니다 :)

> 🤖 봇 시작

```
npm start
```

> 💻 봇 명령어

- !#공지등록 (공지내용) : (공지내용)을 공지로 등록합니다
- !#공지변경 (공지내용) : 공지 내용을 (공지내용)으로 변경합니다

> 📜 sensitive-infos.js 내용 (Root 위치에 반드시 있어야 합니다)
```
export const token = "토큰";
```
